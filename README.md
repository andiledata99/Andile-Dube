Hi, I’m Andile Dube! 👋<br>I’m a Data Analyst transitioning into Data Engineering, focused on building scalable, reliable, and analytics-ready data systems that turn raw data into high-impact business insights.<br>I’ve worked on end-to-end data projects, including designing and implementing ETL pipelines that process 1.2M+ retail transaction records, building data warehouses using star schema modeling, and developing analytical views to support reporting on sales performance, trends, and operational efficiency.<br>My core strength lies in bridging data analysis and data engineering—ensuring data is not only analyzed, but properly ingested, cleaned, structured, and optimized for performance and accuracy.
